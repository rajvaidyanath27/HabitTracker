# 🚀 HabitTracker

A modern habit tracking application that helps users build consistency, maintain daily streaks, and monitor their progress over time.

## ✨ Features

- 🔐 User Authentication
- 📅 Daily Habit Tracking
- 🔥 Streak Management
- 📊 Progress Analytics
- 📈 Habit Statistics
- 🎯 Goal Setting
- 🌙 Clean & Responsive UI
- 💾 Persistent Data Storage

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS / Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT Authentication

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rajvaidyanath27/HabitTracker.git
```

Navigate to the project:

```bash
cd HabitTracker
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
